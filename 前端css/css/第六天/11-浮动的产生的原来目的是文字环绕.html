传说在水中被游鱼豢养大的阿飞（刘德华饰），性格飞扬跳脱，虽然身负奇技， 但漠视名利。 一次机缘，结识了燕十三太子（钟镇涛饰），两人投缘，结成莫逆。燕于飞之渔村养伤之际，突见兵马寻觅至附近，飞率燕等人到一秘密皇陵匿藏，追问之下，始知挚友乃前朝遣裔，正被夺位之十四太子（王霄饰）追杀，并望能给合兰陵国之首 － 兰陵君，共举皇者之师，飞接令前往拯救兰陵君及其女儿月牙儿（梅艳芳饰），两人于逃亡过程中互生情愫，但奈于月牙儿与燕十三有婚约在先，二人不禁怅惘。 三人君子之后，更添情网之交煎，再加上一向心仪燕之护法魔仙儿（张曼玉饰）从中作梗，四人苦在心头。 那边厢，十四太子武艺高强，手足二人战至难分难解，生死未卜 [2] 。
